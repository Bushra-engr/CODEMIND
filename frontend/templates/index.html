About

Built for engineers, by engineers.

CodeMind is an agentic AI assistant powered by LangGraph, LangChain and Groq. It coordinates multiple specialised LLM agents to deliver production-grade output every run.

Tech Stack

FastAPILangGraphLangChain Groq LLMPostgreSQLNeon Cloud JWTVanilla JS Multi AgentPython

AI Workflow

  1. Reviewer — finds bugs, smells, anti-patterns
  2. Bug Fixer — repairs logic & edge cases
  3. Optimizer — improves perf & complexity
  4. Documenter — generates README + inline docs
  5. Explainer — dry-run walkthrough
  6. GitHub Agent — repo + push

Security

JWT-based authentication, scoped tokens, OAuth-secured GitHub push. Your code is never used to train any model.
